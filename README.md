# Kalkulus 2 UAS Helper

Website statis untuk belajar materi UAS Kalkulus 2 dengan bahasa sederhana.

## Isi website

- Cara menentukan angka x dan batas integral
- Logaritma natural
- Integral parsial tentu dan tak tentu
- Luas daerah
- Volume benda putar metode cakram
- Volume benda putar metode cincin
- Volume benda putar metode kulit tabung
- Visualisasi grafik berbasis canvas
- Latihan soal dari kisi-kisi

## Cara membuka

Buka file `index.html` di browser.

Agar rumus LaTeX tampil rapi, koneksi internet diperlukan karena website memakai MathJax CDN.
